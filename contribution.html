Want to add an event to the timeline?


1. Create a Pull Request

Open the "data.js" file and add your event to the "timelineData" array using this format:

{
  "title": "Event Title",
  "date": "YYYY-MM-DD",
  "description": "Brief description of what happened.",
  "source": "https://example.com/source",
  "sourceTitle": "Source Title"
}

Important: Your source must be verifiable. For videos, include the full VOD or an archived version. For tweets, link directly no screenshots.


2. Submit the Pull Request

After editing "data.js", push your changes to the main branch and open a pull request on GitHub.

In your pull request description, please include a short summary of the event.

Your submission will be reviewed and merged if everything checks out.
